# Phase 4.2 — Content Quality Playbook

## Purpose

Briefing #1 must feel personal, useful, and safe. A pass means the message contains at least one user-specific insight, uses cautious language, and offers a concrete follow-up query.

## Editorial Rules

1. Start insight language with **"Bé Tiền nhận thấy..."** instead of absolute advice.
2. Do not write **"Bạn nên..."** in first-briefing insights.
3. Every insight must include at least one of:
   - segment comparison,
   - concrete action suggestion,
   - follow-up question/query.
4. Keep each insight under 200 characters so the full briefing stays scannable.
5. Never reference private exact numbers in operator replies; use portfolio-level wording.

## Operator Review Checklist — First 5 Founding-Member Briefings

For each of the first five briefing #1 sends after Phase 4.2 launch:

- [ ] Contains at least one personalized insight.
- [ ] Uses "Bé Tiền nhận thấy..." cautious framing.
- [ ] Does not use "Bạn nên...".
- [ ] Includes a follow-up query or concrete next action.
- [ ] No exact private asset amount is quoted in operator follow-up.
